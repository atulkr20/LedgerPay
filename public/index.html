// Core Architecture

Built like a bank. Wired like a startup.

Every design decision here is made to handle real-world concurrency, data integrity, and financial auditability at scale.

⊕
Double-Entry Accounting

Every money movement creates immutable debit + credit ledger rows. Balances are derived, never stored. The books always balance.

DEBIT + CREDIT = 0
⚿
Row-Level Locking

SELECT … FOR UPDATE with sorted account IDs. No deadlocks. No phantom reads. Concurrent transfers on the same wallet pair are safely serialized.

PostgreSQL FOR UPDATE
◈
Idempotency Engine

Every state-changing request carries an Idempotency-Key. Redis checks it before any DB hit — retries return the exact same response.

Redis 24h TTL
⬡
ACID Transactions

All money movements run inside prisma.$transaction. Mid-flight failures roll back completely. No partial debits, no ghost credits.

Prisma.$transaction
↺
Financial Immutability

Transactions are never deleted or edited. Refunds create new inverse-entry transactions. Every movement is permanently auditable.

Immutable Ledger
⬟
Type-Safe by Default

Zod validates every request at the boundary. Prisma generates fully typed DB clients. TypeScript enforces it all at compile time.

Zod + Prisma + TS
// How it works

Math that never lies

No stored balance. When you call /balance, PostgreSQL runs a live aggregation across all ledger entries. The result is always exact.

01
Request hits JWT middleware
Bearer token verified, userId extracted
02
Idempotency key checked in Redis
Cache hit → return early. Miss → proceed.
03
Row locks acquired in sorted order
Deadlock-proof via lexicographic ID sort
04
Debit + Credit entries inserted atomically
Single ACID transaction — all or nothing
ledger.service.ts
$ POST /wallets/:id/transfer
→ Idempotency-Key: "txn_8f3acd"
→ Redis: CACHE MISS — proceeding
→ BEGIN TRANSACTION
LOCK acct_abc, acct_xyz // sorted
INSERT DEBIT −₹5,000 // from
INSERT CREDIT +₹5,000 // to
INSERT Transaction record
→ COMMIT ✓ ACID
→ Redis: CACHE result 24h TTL
balance = SUM(credits) − SUM(debits)
₹18,450.00 ← derived, never stored

// Tech Stack

No shortcuts. No abstractions hiding bugs.

Node.js + TypeScript
Core server runtime. Type-safe from HTTP boundary to DB query.
Express.js
Minimal, composable middleware chain. JWT → Idempotency → Route.
PostgreSQL
Primary store. ACID transactions, row-level locking, aggregate balance.
Prisma ORM
Type-safe queries, transaction support, schema-first migrations.
Redis (ioredis)
Idempotency cache with 24h TTL and 10s processing lock.
JWT + bcryptjs
Stateless auth. Hashed passwords. Token-verified middleware.
Zod
Runtime request validation at every endpoint boundary.
Docker
Local PostgreSQL + Redis via compose. One command, full stack.
